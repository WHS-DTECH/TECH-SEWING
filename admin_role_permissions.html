Page Descriptions

  • Home Page: Main Sewing Room activity hub page.
  • Add Activities: Upload and create new practical sewing activities.
  • Browse Activities: View and search the full activity library.
  • Planning: Access planning tools for scheduling and project management.
  • Admin: Administrative functions — user roles, permissions, and system settings.

Important Notes

  • Admin role should always have access to all pages.
  • Public Access is assigned to all logged-out visitors by default.
  • Additional roles are managed in User Role Management.
  • When connected to Google login, changes should apply on the user's next token refresh.
